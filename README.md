# Econia Frontend
## Getting Started
- Pull  [TradingView Submodule](https://github.com/tradingview/charting_library)
```bash
git submodule update
```
- Change directory to frontend folder
```bash
cd src/frontend
```
- Install dependencies
```bash
pnpm i # pnpm is required
```
- Run the development server:
```bash
npm run dev 
```
